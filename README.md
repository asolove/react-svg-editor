React SVG Editor
================

A simple UI widget for creating SVG images. Very very incomplete.

Setup
-----

- `git clone https://github.com/asolove/react-svg-editor.git`
- `cd react-svg-editor`
- `npm install`
- `npm start`
- Open index.html in your browser.
